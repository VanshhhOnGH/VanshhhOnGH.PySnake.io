# 🐍 Snake Game in Python

A classic Snake Game built using Python and Tkinter.

## About the Project
This is a simple Snake Game developed using Python and Tkinter.
It includes snake movement, food generation, collision detection, score tracking, and game-over logic.

## Features
✅ Smooth snake movement  
✅ Food spawning system  
✅ Score tracking  
✅ Collision detection  
✅ GUI using Tkinter  

## Tech Stack
- Python
- Tkinter
> Note: Tkinter usually comes pre-installed with Python.

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/VanshhhOnGH/VanshhhOnGH.PySnake.io.git
```

### 2. Open project folder

```bash
cd VanshhhOnGH.PySnake.io
```

### 3. Run the game

```bash
python snake.py
```

If `python` doesn’t work, try:

```bash
python3 snake.py
```
